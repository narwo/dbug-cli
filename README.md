## D'Bug CLI

```
Command-line tool for D'Bug

Usage: dbug [OPTIONS] <COMMAND>

Commands:
  list    List probes
  reset   Reset target
  status  Target status
  unlock  Unlock target, erases all
  lock    Lock target
  help    Print this message or the help of the given subcommand(s)

Options:
  -v          More verbose output
  -h, --help  Print help
```
